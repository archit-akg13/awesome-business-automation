## 🤖 AI Coding Assistants

AI-powered coding partners that understand context, autocomplete intelligently, and ship code 2-3x faster.

### **Claude Code** ⭐ TOP PICK
- **[Claude Code](https://www.anthropic.com/claude/code)** — Terminal as your AI coding partner. Think in pseudocode → Claude writes, tests, and debugs for you. Uses Claude Sonnet 4.6 (5x cheaper than Opus, same benchmarks).
- 🏷️ **Pricing:** Free (Claude.ai), Paid (Claude Pro, Teams)
- 👍 **Best for:** Full-stack development, rapid prototyping, debugging production code
- **Why it wins:** Understands file structure across your repo. Can access your terminal directly. No context switching. Fastest workflow.

### **GitHub Copilot**
- **[GitHub Copilot](https://github.com/features/copilot)** — AI pair programmer integrated into your IDE. IntelliSense-powered, trained on public code.
- 🏷️ **Pricing:** $10/month (individual), $19-39/user/month (enterprise)
- 👍 **Best for:** Individual developers, IntelliJ/VS Code workflows
- **Why it's here:** Industry standard. Works offline. VSCode integration is seamless.

### **Cursor**
- **[Cursor](https://www.cursor.com)** — VS Code fork with AI baked in. Tab to autocomplete, Ctrl+K to write blocks, @symbols to reference files.
- 🏷️ **Pricing:** Free tier, $20/month (Cursor Pro)
- 👍 **Best for:** AI-first developers, entire codebase indexing
- **Why it's here:** Best for long-form code generation. Understands your whole project at once.

### **Windsurf by Codeium**
- **[Windsurf](https://codeium.com/windsurf)** — Agentic AI editor. Autonomous code generation, file search, terminal access integrated.
- 🏷️ **Pricing:** Free tier, $40/month (Pro)
- 👍 **Best for:** Larger projects, autonomous agents
- **Why it's here:** Multi-file editing, true agentic behavior (not just autocomplete).

### **Cline**
- **[Cline](https://github.com/cline/cline)** — Open-source VS Code extension. Claude-powered, controlled terminal access, web browsing.
- 🏷️ **Pricing:** FREE (open-source)
- 👍 **Best for:** Budget-conscious teams, self-hosted deployments
- **Why it's here:** Full autonomy over AI behavior, no vendor lock-in, can run locally.

### **Amazon CodeWhisperer**
- **[AWS CodeWhisperer](https://aws.amazon.com/codewhisperer/)** — AWS's IDE-agnostic AI coding assistant. Security scan included.
- 🏷️ **Pricing:** Free tier, $19/month (individual)
- 👍 **Best for:** AWS-heavy teams, security-conscious enterprises
- **Why it's here:** Native IAM integration, code security scanning built-in.

## ⚙️ Workflow Automation Platforms

**The backbone of business automation.** These platforms connect your apps, trigger workflows, and eliminate manual tasks.

### **n8n** ⭐ TOP PICK FOR INDIAN TEAMS
- **[n8n](https://n8n.io)** — Open-source, self-hosted workflow automation. 500+ integrations, visual builder, AI-powered nodes.
- 🏷️ **Pricing:** FREE self-hosted, €20-840/month cloud
- 👍 **Best for:** Cost-conscious teams, complex enterprise workflows, Indian startups
- **Why it wins:** 1/10th the price of Zapier. Can deploy on your own servers. Full control.
- **📌 Real case:** Saved a client **₹12,000/year** by replacing Zapier with n8n self-hosted.

### **Make.com** (formerly Integromat)
- **[Make.com](https://www.make.com)** — Powerful visual workflow builder. 1000+ apps, complex branching logic, strong community.
- 🏷️ **Pricing:** $10-990/month
- 👍 **Best for:** Mid-market teams, complex workflows needing human-readable logic

### **Zapier**
- **[Zapier](https://zapier.com)** — Industry standard. 6000+ integrations, easy setup, trusted by enterprises.
- 🏷️ **Pricing:** $29-600+/month
- 👍 **Best for:** Non-technical teams, quick integrations
- **⚠️ Warning:** Expensive at scale. Consider n8n or Make for cost optimization.

### **Activepieces**
- **[Activepieces](https://www.activepieces.com)** — Modern, open-source no-code automation. Self-hosted, competitive pricing.
- 🏷️ **Pricing:** FREE open-source, $99/month cloud
- 👍 **Best for:** Teams wanting modern open-source alternative to Zapier

## 🧠 AI Agent Frameworks

Build autonomous AI agents that think, act, and iterate. From simple chains to multi-agent systems.

### **Claude MCP (Model Context Protocol)** ⭐ EMERGING STANDARD
- **[Claude MCP](https://modelcontextprotocol.io)** — USB port for AI agents. Connect Claude to any data source, tool, or system.
- 🏷️ **Pricing:** FREE protocol, tools vary
- 👍 **Best for:** Building autonomous Claude-powered agents
- **Why it wins:** Fastest growing protocol. Simple, scalable. Vendors building integrations rapidly.

### **LangChain**
- **[LangChain](https://python.langchain.com)** — Build chains of LLM calls. Agents, memory, document processing, structured outputs.
- 🏷️ **Pricing:** FREE (open-source), LangSmith monitoring available
- 👍 **Best for:** Production LLM applications, chaining complex workflows

### **LangGraph**
- **[LangGraph](https://langgraph.js.org)** — State machines for multi-step agent workflows. Deterministic, debuggable, human-in-the-loop.
- 🏷️ **Pricing:** FREE (open-source)
- 👍 **Best for:** Complex, long-running agent tasks

### **CrewAI**
- **[CrewAI](https://crewai.com)** — Cooperative multi-agent teams. Agents collaborate to solve complex tasks.
- 🏷️ **Pricing:** FREE (open-source), enterprise support available
- 👍 **Best for:** Multi-agent systems, research automation

### **Microsoft AutoGen**
- **[Microsoft AutoGen](https://microsoft.github.io/autogen/)** — Multi-agent conversation framework. Agents talk to each other to solve tasks.
- 🏷️ **Pricing:** FREE (open-source)
- 👍 **Best for:** Conversation-based agent workflows

### **Vercel AI SDK**
- **[Vercel AI SDK](https://sdk.vercel.ai)** — Build AI applications with TypeScript. Streaming, function calling, multi-model support.
- 🏷️ **Pricing:** FREE (open-source)
- 👍 **Best for:** Next.js teams, streaming LLM responses

## 💰 Cost Optimization & Monitoring

Reduce AI/API costs, monitor spend, optimize LLM usage. Critical for scaling automation profitably.

### **LiteLLM Proxy**
- **[LiteLLM](https://docs.litellm.ai)** — Unified LLM API. Route between Claude, GPT, Gemini. Track costs per model.
- 🏷️ **Pricing:** FREE (open-source)
- 👍 **Best for:** Optimizing AI API costs, multi-model switching

### **Cost Optimization Tips** 📋
1. **Use Sonnet instead of Opus** — 5x cheaper, same benchmarks on most tasks
2. **Batch API calls** — Anthropic offers 50% discount on batch processing
3. **Self-host n8n/Activepieces** — Avoid per-workflow pricing of Zapier/Make
4. **Cache prompts** — Claude's prompt caching saves 90% on repeated contexts
5. **Monitor token usage** — Set up billing alerts on Claude/OpenAI dashboards

- **📌 Real case:** Client saved **₹85,000/month** by switching from Opus to Sonnet for same benchmarks.

## 🎯 How to Choose Your Stack

### **By Budget** 💰

| Budget | Recommended Stack |
|--------|-------------------|
| **$0** (bootstrapped) | n8n self-hosted + LangChain + Activepieces + Retool self-hosted |
| **₹0-25K/month** | n8n cloud + Claude API + Cline extension |
| **₹25-50K/month** | Make.com + Zapier (small workflows) + Claude Code + Cursor |
| **₹50K+/month** | Power Automate + UiPath + dedicated ops team |

### **By Team Size** 👥

| Team Size | Best Fit |
|-----------|----------|
| **Solo founder** | Claude Code + n8n + Activepieces |
| **1-5 people** | Make.com + Zapier + Bubble for MVPs |
| **5-50 people** | Power Automate + Retool + n8n |
| **50+ people** | UiPath + Automation Anywhere + enterprise workflows |

### **By Use Case** 📌

| Use Case | Recommended |
|----------|-------------|
| **Save on AI API costs** | LiteLLM Proxy + Claude Sonnet + batch API |
| **Build internal tools** | Retool or Appsmith (2-3 hours vs 2-3 weeks) |
| **Automate customer workflows** | Zapier / Make.com / n8n |
| **Build AI agents** | LangChain + LangGraph + Claude Code |
| **Scale data pipelines** | Airflow + dbt |
